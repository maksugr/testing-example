# ReDI Munich: Testing example

## How to get the application

```bash
git clone git@github.com:maksugr/testing-example.git
cd testing-example
npm i
```

## How to run the application

```bash
npm run dev
```

## How to test the application

```bash
npm test
```

### unit tests

```bash
npm run vitest
```

### e2e tests

```bash
npm run cy:open
```
